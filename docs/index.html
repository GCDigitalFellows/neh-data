Visualizing NEH

This project is part of a collaboration by several Digital Fellows of the Graduate Center Digital Initiatives. Our efforts were initiated in light of the recently proposed budget by the Trump administration that called for the elimination of the National Endowment of Humanities (NEH) and the National Endowment of the Arts (NEA).

All the code for this project are available on Github at GCDigitalFellows/visualizing-neh. If you have any issues with the data or presentation, please submit issues through Github.
